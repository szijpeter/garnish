# Security Policy

## Supported Versions

Garnish is pre-1.0. Security fixes are provided on the latest branch (`main`).

## Reporting a Vulnerability

Please use GitHub private vulnerability reporting:

- https://github.com/szijpeter/garnish/security/advisories/new

Please do not open public issues for security vulnerabilities.

When reporting, include:

- affected module and version
- platform (Android/iOS)
- impact and attack scenario
- minimal reproduction steps

We will acknowledge reports as quickly as possible and coordinate a fix and disclosure.
