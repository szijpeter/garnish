# Garnish Brand Mark

## Files
- `garnish-mark.svg`: canonical vector source.
- `garnish-mark-1024.png`: export for app stores, social cards, and docs.

## Usage
- Keep at least 8% clear space around the mark on all sides.
- Minimum display size: 24px for UI icon usage, 64px for brand lockups.
- Prefer full-color mark on light neutral backgrounds.
- For monochrome contexts, use a single-color silhouette of the sprig shape.

## Palette
- Leaf gradient: `#4f8a5d` to `#2f5b3c`
- Neutral background gradient: `#f5f2e9` to `#ece7d8`
